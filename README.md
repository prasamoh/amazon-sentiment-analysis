# Amazon Sentiment Analysis (Capstone Project)

## Data Collection

Data was collected from X (Twitter) using Apify's tweet-scraper.

### Query
(Amazon OR @amazon OR #amazon) lang:en

### Time Period
January 2026 – March 2026

### Methodology
- Data collected in batches using date-based filtering
- Replies excluded to reduce noise
- API-based scraping via Apify

### Files
- raw JSON data: data/raw/
- cleaned CSV: data/raw/
